## What We Check

### 1. üìã Photo Information (Metadata)
Every photo contains hidden data about:
- Camera model and settings
- Date and time taken
- Software used
- GPS location (if available)

**What we look for**: Missing data, inconsistent timestamps, signs of editing software

### 2. üé® Compression Levels (ELA)
When you edit and re-save a photo, edited areas get compressed differently than original areas.

**What we look for**: Bright spots in the ELA image showing different compression levels

### 3. üîä Digital Noise
Real cameras create consistent "noise" patterns across a photo. Editing disrupts these patterns.

**What we look for**: Inconsistent noise that suggests copy-paste or retouching

### 4. ü§ñ AI Detection
Our AI analyzes thousands of subtle patterns humans can't see.

**What we look for**: Machine learning patterns that indicate manipulation

## Common Questions

### "Why did my real photo fail?"
Several reasons:
- The photo was already edited (filters, cropping, etc.)
- It was sent through messaging apps that compress it
- It's a screenshot
- It was downloaded from social media (they modify photos)

### "Why did an edited photo pass?"
- Very subtle professional edits are hard to detect
- The editing was done before the first save
- The entire image was processed (no inconsistencies)

### "What if I'm not sure about the result?"
- Check the technical details (click "Show Technical Details")
- Try uploading the original file
- Consider the confidence score
- For important cases, consult a professional forensic expert

## Troubleshooting

**Solution**: Use JPEG or PNG files only

### Error: "Analysis failed"
**Solution**: 
1. Try a different photo format

3. Refresh the page and try again

### Results seem wrong
**Possible causes**:
1. Photo was already edited
2. Screenshot instead of original
